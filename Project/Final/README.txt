About The Program:-
................................................

>The Dictionary contains information about animals and birds present in IIT Mandi.
>There is a data file which contains information about the animals and birds (animal.txt).
>User can get the information of their desired animals by various methods in our program.



................................................
General Instruction For User:-
................................................
>Open your compiler and run our program or open our main.c file directly in your system.
>There are many files in the program, please do not move it out from the folder, it may lead terminating the program.
>Do not change the raw data of the program it may cause wrong information.
>If some part of our program don't work in your compiler please try to update your compiler.
>We will be happy to get any suggestions regarding our program/code. 



................................................
Input:-
................................................
>We have multiple methods for the user to search their desired animals and access their information.
>User can directly enter the name or choose a species category or can see the list of entire animals and birds.
>User can directly access the help function from the main page of program.
 

................................................
Exception handling:-
................................................
>Shows error message on typing invalid integer or cahracter input (typing integer in place of character or vice-versa, entering name of species which is not present in dictionary).
>Asks the user again for the correct input for any incorrect input.
